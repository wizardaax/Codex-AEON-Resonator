## Overview

**Codex-AEON-Resonator** is a research and analysis repository built on the **Recursive Field Framework (RFF)** — a cross-domain pattern recognition methodology founded on the core formula:

```
r = a√n,  θ = nφ
```

Where `φ` is the golden ratio (≈1.6180339887). This formula generates recursive polar coordinates that expose deep topological equivalence across disparate domains — from ancient civilisational structures to undeciphered manuscripts.

This repository documents the full analytical thread: from a single dynastic anomaly in 14th-century Ethiopia, through 12,000 years of extraction topology, to the first quantitative morphological case for Ethiopian origin of the Voynich Manuscript.
